**Credential Helper:** এটি মূলত গিট (Git)-কে নির্দেশ দেয় যে, আপনি একবার যে পাসওয়ার্ড (এখানে টোকেন) দিবেন, সেটি যেন আপনার VPS-এ একটি ফাইলে সেভ করে রাখে। ফলে ভবিষ্যতে আর কখনো পাসওয়ার্ড চাইবে না।

এটি সফলভাবে করার জন্য আপনাকে নিচের ধাপগুলো অনুসরণ করতে হবে:

### ধাপ ১: GitHub থেকে Personal Access Token (PAT) তৈরি করা

যেহেতু গিটহাব এখন আর সাধারণ একাউন্ট পাসওয়ার্ড সাপোর্ট করে না, তাই প্রথমে আপনাকে একটি টোকেন বানাতে হবে (যদি আগে বানানো না থাকে)।

১. আপনার কম্পিউটার বা ফোনের ব্রাউজার দিয়ে [GitHub.com](https://github.com)-এ লগইন করুন।
২. ডানপাশের উপরের কোণায় আপনার প্রোফাইল ছবিতে ক্লিক করে **Settings**-এ যান।
৩. বামদিকের মেনুর একদম নিচে স্ক্রল করে **Developer settings**-এ ক্লিক করুন।
৪. এরপর বামদিক থেকে **Personal access tokens**-এ ক্লিক করে **Tokens (classic)** নির্বাচন করুন।
৫. ডানদিকে **Generate new token** বাটনে ক্লিক করে **Generate new token (classic)** নির্বাচন করুন (পাসওয়ার্ড চাইতে পারে, দিলে তা দিন)।
৬. **Note** বক্সে একটি নাম দিন (যেমন: "My Ubuntu VPS")।
৭. **Expiration** অপশন থেকে মেয়াদের দিন সিলেক্ট করুন। (আপনি চাইলে "No expiration" দিতে পারেন, তাহলে এই টোকেনটি কখনোই বাতিল হবে না)।
৮. **Select scopes** এর নিচে **`repo`** লেখা বক্সে টিক দিন (এটি খুবই গুরুত্বপূর্ণ, নাহলে প্রাইভেট রিপো ক্লোন করা যাবে না)।
৯. একদম নিচে গিয়ে **Generate token** বাটনে ক্লিক করুন।
১০. এখন স্ক্রিনে `ghp_` দিয়ে শুরু হওয়া একটি লম্বা কোড দেখতে পাবেন। **কোডটি কপি করে নিন** (এটি আপনার নতুন পাসওয়ার্ড হিসেবে কাজ করবে)।

---

### ধাপ ২: VPS-এ Credential Helper চালু করা

এখন আপনার Ubuntu VPS-এর টার্মিনালে যান এবং নিচের কমান্ডটি রান করুন:

```bash
git config --global credential.helper store

```

*এই কমান্ডটি কোনো আউটপুট দেখাবে না, তবে এটি গিট-কে বলে দিলো যে এরপর যে পাসওয়ার্ড দেওয়া হবে তা যেন সেভ করে রাখা হয়।*

---

### ধাপ ৩: রিপোজিটরি ক্লোন করে পাসওয়ার্ড সেভ করা

এখন আপনি আপনার প্রাইভেট রিপোজিটরিটি সাধারণ HTTPS লিংক দিয়ে ক্লোন করার চেষ্টা করুন:

```bash
git clone https://github.com/আপনার-ইউজারনেম/আপনার-রিপোজিটরির-নাম.git

```

এটি রান করার পর আপনার কাছে Username এবং Password চাইবে:

* **Username:** আপনার গিটহাবের ইউজারনেমটি লিখুন এবং `Enter` দিন।
* **Password:** এইখানে আপনার **কপি করা সেই টোকেনটি পেস্ট করুন** এবং `Enter` দিন। (লুকানো নিরাপত্তার জন্য টার্মিনালে পেস্ট করলে কিছু দেখা যায় না, তাই একবার পেস্ট করে সরাসরি Enter চাপবেন)।

### কাজ শেষ!

টোকেনটি সঠিক হলে আপনার প্রাইভেট রিপোজিটরিটি ক্লোন হয়ে যাবে। **এবং সবচেয়ে মজার বিষয় হলো**, Credential Helper এখন আপনার ইউজারনেম এবং টোকেনটি আপনার সার্ভারে সেভ করে ফেলেছে।

এরপর থেকে আপনি যতবারই `git clone`, `git pull` বা `git push` করবেন, গিটহাব আর কখনোই আপনার কাছে পাসওয়ার্ড চাইবে না!

> **একটি ছোট্ট নোট:** এই পদ্ধতিতে গিট আপনার টোকেনটি সার্ভারের `~/.git-credentials` নামক একটি ফাইলে টেক্সট আকারে সেভ করে রাখে। যেহেতু এটি আপনার নিজের VPS, তাই এটি সম্পূর্ণ নিরাপদ।

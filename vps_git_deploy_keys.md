# শুধুমাত্র একটি নির্দিষ্ট রিপোজিটরির এক্সেস দেওয়া (GitHub Deploy Keys)

যদি আপনি কোনো কোম্পানির VPS-এ কাজ করেন এবং চান না যে আপনার সমস্ত ব্যক্তিগত প্রোজেক্ট বা অন্যান্য রিপোজিটরির অ্যাক্সেস ওই VPS-এ থাকুক (কারণ Personal Access Token ব্যবহার করলে আপনার অ্যাকাউন্টের সব রিপোজিটরির অ্যাক্সেস চলে যায়), তাহলে **Deploy Keys** ব্যবহার করা সবচেয়ে নিরাপদ ও উত্তম মাধ্যম। এতে ওই সার্ভারটি শুধুমাত্র একটি নির্দিষ্ট রিপোজিটরিই রিড বা রাইট করতে পারবে।

#### ধাপ ১: VPS-এ একটি SSH Key তৈরি করা

১. আপনার VPS-এর টার্মিনালে প্রবেশ করে নিচের কমান্ডটি রান করুন:
   ```bash
   ssh-keygen -t ed25519 -C "your-email@example.com"
   ```
   এখানে `"your-email@example.com"` হলো একটি কমেন্ট বা লেবেল যা দিয়ে চেনা যায় চাবিটি কার। আপনি এখানে আপনার গিটহাবের ইমেইল, ইউজারনেম, অথবা সার্ভারের নাম (যেমন: `"my-vps"`) যা খুশি দিতে পারেন। এমনকি চাইলে `-C` অংশটি বাদও দিতে পারেন। এরপর ফাইল সেভ করার লোকেশন ও পাসফ্রেজের অপশন আসলে `Enter` চেপে ডিফল্ট রেখে দিন।

২. এবার তৈরি হওয়া পাবলিক কী (Public Key) টি দেখতে নিচের কমান্ডটি দিন এবং পুরো কোডটি কপি করুন:
   ```bash
   cat ~/.ssh/id_ed25519.pub
   ```
   *(কোডটি দেখতে সাধারণত `ssh-ed25519 AAAAC3Nz... your-email@example.com` এমন হবে।)*

#### ধাপ ২: GitHub-এ Deploy Key যুক্ত করা

১. গিটহাবে গিয়ে আপনার নির্দিষ্ট রিপোজিটরিটিতে যান (যেটির অ্যাক্সেস দিতে চান)।

২. রিপোজিটরির উপরের মেনু থেকে **Settings**-এ যান।

৩. বামদিকের মেনু থেকে **Deploy keys** অপশনে ক্লিক করুন।

৪. ডানদিকের উপরে **Add deploy key** বাটনে ক্লিক করুন।

৫. **Title** বক্সে একটি নাম দিন (যেমন: "Company VPS Access")।

৬. **Key** বক্সে আপনার কপি করা পাবলিক কী (Public Key) টি পেস্ট করে দিন।

৭. **Allow write access** বক্সে টিক চিহ্ন দিন (যদি আপনি ওই সার্ভার থেকে কোড পুশ করতে চান, অন্যথায় শুধু কোড রান/পুল করার জন্য এটি আনচেক রাখতে পারেন)।

৮. **Add key** বাটনে ক্লিক করুন।

#### ধাপ ৩: SSH লিংক দিয়ে ক্লোন করা
এখন VPS-এ ক্লোন করার জন্য **HTTPS** লিংকের পরিবর্তে **SSH** লিংক ব্যবহার করুন:
```bash
git clone git@github.com:আপনার-ইউজারনেম/আপনার-রিপোজিটরির-নাম.git
```
ব্যাস! এখন ওই VPS থেকে শুধুমাত্র এই নির্দিষ্ট রিপোজিটরিটিই অ্যাক্সেস করা যাবে এবং আপনার অ্যাকাউন্টের অন্য কোনো প্রোজেক্টে কেউ অ্যাক্সেস পাবে না।
